# ✦ My Tasks — Todo App

A clean, modern task management app built with **React** and **Vite**. Manage your daily tasks with style — no backend, no account required.

🔗 **Live Demo → [my-todo-app-001.netlify.app](https://my-todo-app-001.netlify.app/)**

---

## Features

- ✅ Add, edit, and delete tasks
- 🔄 Track status — **Todo**, **In Progress**, **Completed**
- 🔍 Search tasks by title or description
- 🏷 Filter by status with live counts
- 💾 Persists data in `localStorage` — survives page refreshes
- 📱 Fully responsive — mobile bottom sheet, desktop modal, floating action button on small screens

---

## Tech Stack

| Tool | Purpose |
|---|---|
| [React](https://react.dev) | UI framework |
| [Vite](https://vitejs.dev) | Build tool & dev server |
| CSS-in-JS (inline styles) | Styling — no external CSS framework needed |
| localStorage | Client-side data persistence |
| [Netlify](https://netlify.com) | Hosting & deployment |

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/my-todo-app.git

# Navigate into the project
cd my-todo-app

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open **http://localhost:5173** in your browser.
